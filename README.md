# 基于 Buck 电路的 PID 与 Fuzzy 控制设计

## 📖 项目简介
本项目是“智能控制基础”课程设计，主要内容是对 Buck 降压斩波电路进行参数计算、建模，并利用 PID 控制和模糊控制（Fuzzy Control）两种方法实现闭环控制，以满足特定的动态和静态响应指标。

通过 MATLAB/Simulink 进行仿真验证，对比了两种控制策略在不同输入电压和负载扰动下的性能表现。

## 📂 项目文件说明

仓库内主要包含以下文件：

* **仿真模型**
    * `*.slx`: MATLAB/Simulink 仿真文件，包含了 Buck 电路的电路模型及 PID 控制器（基于频域法校正）和 模糊控制器 (Fuzzy Controller)（基于二维模糊规则）的设计。

* **实验报告**
    * `*.tex`: 实验报告的 LaTeX 源代码。包含了封面、正文及排版设置（已修复下划线对齐问题）。

## 🚀 环境与使用

### 1. 运行仿真
* **软件需求**: MATLAB / Simulink (本模型构建使用 R2024b 完成，R2025 可能导致部分关键库缺失因此不建议使用)
* **步骤**:
    1.  使用 MATLAB 打开 `fuzzy_pid_controller.slx`，`pid_controller.slx` 文件。
    2.  等待模型加载完毕后，点击 "Run" 按钮开始仿真。
    3.  通过 Scope 观察输出波形。

### 2. 编译实验报告
* **编译环境**: TeXLive / Overleaf / MacTeX
* **编译方式**: XeLaTeX (推荐，以支持中文)
